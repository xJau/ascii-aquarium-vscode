# Changelog

All notable changes to the ASCII Aquarium extension are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-31

First public release.

### Added
- Live animated ASCII aquarium in a VS Code webview panel and Explorer sidebar.
- 12 fish species with randomized ANSI colors swimming left and right.
- Rising bubbles, swaying seaweed (with respawn), and a decorative castle.
- Five random visitors: shark, whale, ship, big fish, and a deep-lurking monster.
- Click-to-feed: drop food and watch nearby fish chase and eat it.
- Toolbar controls: pause/resume, speed up/slow down, restart.
- Status-bar **🐟 Aquarium** toggle and `ASCII Aquarium: Toggle` command.
- Configuration: `asciiAquarium.targetFps`, `asciiAquarium.enabledEvents`,
  `asciiAquarium.feedEnabled`.
